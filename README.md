DesktopYouTube
==============

Converts your offline music into a youtube playlist.
Try out the app at :  http://mukherjeerohit93.azurewebsites.net/

Stack Used:
==============
<ol>
<li>Metro Bootstrap</li>
<li>JQuery</li>
<li>PHP</li>
<li>MYSQL</li>
<li>Youtube API V2.0</li>
</ol>
